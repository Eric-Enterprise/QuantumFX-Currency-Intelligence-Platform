## 📦 INSTALLATION & SETUP

### **Prerequisites**
- Python 3.13+
- pip package manager
- 50MB free disk space
- 1GB RAM (minimum)
- Internet connection (for live rates)

### **Installation Steps**

```bash
# Clone the repository
git clone https://github.com/Eric-Enterprise/QuantumFX-Currency-Intelligence-Platform.git
cd QuantumFX-Currency-Intelligence-Platform

# Create virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install requests

# Launch the Premium Engine
python QuantumFX_Premium_Engine.py
```

### **Configuration**

Edit the `Config` class in the Python file to customize:

```python
# API Configuration
PRIMARY_API = "https://api.frankfurter.app"
CACHE_EXPIRY_SECONDS = 3600

# UI Configuration
WINDOW_WIDTH = 1600
WINDOW_HEIGHT = 1000
MIN_WIDTH = 1200
MIN_HEIGHT = 800

# Theme Colors (fully customizable)
ACCENT_COLOR = "#00D9FF"
BG_PRIMARY = "#0A0E27"
```
